# HTML-NOTES

## Labels 
Used to create HTML elements
1. Headers Label
	All the way up from "h1" to "h6"
	```html
	<h1>This is the title 1</h1>
	```
1. Paragraph Label
	```html
	<p>This is a paragraph</p>
	```
1. Lists
	1. Ordered List
		```html
		<ol>
			<li>List Element</li>
		</ol>
		```
	1. Unordered List
		```html
		<ul>
			<li>List Element</li>
		</ul>
		```
1. Nested Elements
	Defined inside other HTML elements.
	```html
	<p>This is a paragraph with a <strong>nested element</strong>.</p>
	```
